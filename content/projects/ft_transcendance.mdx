<span style={{ display: 'block', textAlign: 'center', fontSize: '34px'}}><u><i>**ft_transcendance**</i></u></span>

<span style={{textAlign: 'center'}}>
ft_transcendance is a <strong>Bachelor’s final-year project</strong> offering a secure, multiplayer web gaming experience.<br />
Built with <strong>Django</strong>, <strong>PostgreSQL</strong>, and <strong>Bootstrap</strong>, it features <strong>user account management</strong> (anonymization, 2FA, deletion) and multiplayer games like Pong and Tic-Tac-Toe,
including an 8-player <strong>tournament mode</strong>.<br /> Players can also challenge a <strong>Python-coded AI</strong> in Pong, with game history tracking included.
</span>


<hr style={{ border: '0.1px solid black' }} />

<br />


<span style={{ display: 'block', textAlign: 'center', fontSize: '28px' }}>**Project Overview**</span>


- <span style={{fontSize: '20px' }}><u>Tech Stack :</u></span>
  - <strong>Django</strong> : The application’s backend is powered by Django, providing efficient management of <strong>server-side</strong> operations.
  - <strong>Bootstrap</strong> : The frontend is built using Bootstrap Toolkit, providing a <strong>responsive</strong> and intuitive user interface.
  - <strong>PostgreSQL</strong> : Utilizes PostgreSQL for reliable data storage and management, ensuring <strong>stability</strong> in all backend processes.

- <span style={{fontSize: '20px' }}><u>Gameplay Features :</u></span>
  - <strong>AI Opponent</strong>: Allows users to play Pong against an <strong>AI-controlled</strong> opponent.
  - <strong>Multiplayer Mode</strong>: Supports <strong>real-time</strong> multiplayer games, including Pong and Tic-Tac-Toe.
  - <strong>Tournament System</strong>: Includes a tournament feature for up to <strong>8 players</strong>.

- <span style={{fontSize: '20px' }}><u>Authentication and Security :</u></span>
  - <strong>Remote Authentication</strong> : Integrates OAuth 42 for secure authentication through the <strong>42 intra API</strong>, ensuring reliable user verification.
  - <strong>Two-Factor Authentication (2FA) & JWT</strong> : Strengthens security with two-factor authentication and <strong>JSON Web Tokens</strong> for session management and user verification.

___

<span style={{ display: 'block', textAlign: 'center', fontSize: '28px' }}>**Additional Features**</span>

- <span style={{fontSize: '20px' }}><u>Privacy and Compliance :</u></span>
  - <strong>GDPR Compliance</strong> : Includes options for GDPR compliance, such as user <strong>anonymization</strong>, local data management, and account <strong>deletion</strong>, ensuring user <strong>privacy</strong> and <strong>data protection</strong>.

- <span style={{fontSize: '20px' }}><u>Browser Compatibility :</u></span>
  - <strong>Multi-Browser Support</strong> : Extended compatibility across major web browsers to provide a <strong>seamless experience</strong> on various platforms.
