**<u>ft_transcendance</u>**

ft_transcendance is an advanced web application designed to deliver an engaging and secure gaming experience. The project is built using Django for the backend framework, ensuring a robust and scalable infrastructure. A highlight of the application is its AI-controlled opponent for a classic Pong game, providing users with a challenging gameplay experience.

**Project Overview:**

Backend Framework: The application leverages Django, a powerful backend framework, to manage server-side operations efficiently.

AI Opponent: Users can play Pong against an AI-controlled opponent, enhancing the gameplay with intelligent responses.

Remote Authentication: Implementing OAuth 42, the application ensures secure remote authentication, integrating seamlessly with the 42 intra API for user verification.

Two-Factor Authentication (2FA) and JWT: To enhance security, ft_transcendance includes two-factor authentication and JSON Web Tokens for secure session management and user verification.

Additionally, the project incorporates several minor yet significant features:

Database Management: Utilizing PostgreSQL, the application ensures reliable data storage and management for all backend processes.

GDPR Compliance: The application includes options for GDPR compliance, such as user anonymization, local data management, and account deletion, ensuring user privacy and data protection.

Browser Compatibility: Efforts have been made to extend the application's compatibility across various web browsers, ensuring a seamless experience for all users.

Frontend Framework: The frontend is built using a modern framework or toolkit, providing a responsive and intuitive user interface.
